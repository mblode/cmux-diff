# Installation

## Run without installing

```bash
npx diffhub@latest
```

## Global install

```bash
npm install -g diffhub
```

Then run from any git repository:

```bash
diffhub
```

## Requirements

- Node.js 20.11 or later
- A git repository with at least one commit

## Verify installation

```bash
diffhub --version
```
