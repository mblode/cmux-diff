# File Tree

## File sidebar

The left sidebar shows all changed files with per-file `+`/`-` stats, grouped into a collapsible folder tree.

## Filtering

Press `/` or `t` to focus the filter input. Type to narrow the file list — matches against file paths.

## Navigation

- Press `j` to move to the next file
- Press `k` to move to the previous file
- Click a file to scroll to its diff

## Collapsing

- Press `c` to collapse or expand the selected file's diff
- Press `Shift+C` to collapse all files
- Press `Shift+E` to expand all files
