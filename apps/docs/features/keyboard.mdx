# Keyboard Shortcuts

| Key | Action |
| --- | ------ |
| `j` | Next file |
| `k` | Previous file |
| `s` | Toggle split / unified view |
| `/` or `t` | Focus file filter |
| `r` | Refresh diff |
| `c` | Collapse / expand current file |
| `Shift+C` | Collapse all files |
| `Shift+E` | Expand all files |
